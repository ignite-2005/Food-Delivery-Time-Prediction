# 🍔 Food Delivery Time Prediction
**Predictive Modeling using Regression & Classification**

## 📌 What I did in this project:
- **Linear Regression:** Used to predict the continuous value of delivery time in minutes.
- **Logistic Regression:** Used to classify deliveries into "Fast" or "Delayed".
- **EDA:** Performed outlier detection using Boxplots and correlation analysis.

## 📈 Next Steps (Currently Learning):
- Now that I have mastered the basics, I plan to implement **SVM (Support Vector Machines)** to see if it provides better accuracy for the delay classification than Logistic Regression.
